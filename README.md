# 🔇 Smart Silence Remover for Premiere Pro

> AI-powered silence and pause removal for Adobe Premiere Pro.  
> Edit faster, cleaner, and smarter with automatic silence detection and ripple deletion.

---

## 📌 Overview

**Smart Silence Remover** is a professional Adobe Premiere Pro extension designed to automatically detect and remove silent gaps from your timeline.

Built for:

- 🎬 Video Editors
- 🎙 Podcasters
- 📹 YouTubers
- 🎓 Course Creators
- 🎤 Interview Editors

The extension uses advanced audio analysis powered by **FFmpeg** to intelligently identify silence while keeping Premiere Pro responsive during processing.

# 🛠 Requirements

## 1. FFmpeg

This extension requires **FFmpeg** for audio analysis.

### Download FFmpeg

- https://ffmpeg.org

### Add FFmpeg to PATH

Make sure FFmpeg's `bin` folder is added to your system environment variables.

Example:

```bash
C:\ffmpeg\bin
```

---

## 2. Adobe Premiere Pro

Compatible with:

- Adobe Premiere Pro 2022+
- CEP Extensions Environment

---

# 📦 Installation

## Windows

Copy the extension folder to:

```bash
C:\Users\<YourUser>\AppData\Roaming\Adobe\CEP\extensions\
```

---

## macOS

Copy the extension folder to:

```bash
~/Library/Application Support/Adobe/CEP/extensions/
```

---

## Launch Extension

1. Restart Premiere Pro
2. Open:

```text
Window → Extensions → Smart Silence Remover
```

---

# 🚀 Usage

## Step 1 — Open a Sequence

The extension automatically detects your active Premiere Pro sequence.

---

## Step 2 — Configure Settings

Adjust:

- Silence Threshold
- Padding
- Minimum Silence Duration

Example:

```text
Threshold: -35dB
Padding: 120ms
```

---

## Step 3 — Analyze Audio

Click:

```text
Analyze Audio
```

The extension will generate a visual Audio Map.

---

## Step 4 — Review Silence Zones

- Browse detected silence areas
- Keep important pauses
- Disable zones manually if needed

---

## Step 5 — Remove Silences

Click:

```text
Remove Silences
```

The extension will:

- Process cuts in real-time
- Ripple-delete silence gaps
- Maintain sync across tracks

---

## Step 6 — Undo Anytime

Made a mistake?

Use:

```bash
Ctrl + Z
```

or the built-in Undo button.
